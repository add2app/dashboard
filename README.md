## Color Usage Guidelines

### Primary Color Usage
- Use **Primary Red** (`#E63946`) for:
  - Main action buttons
  - Active navigation items
  - Important metrics and highlights
  - Chart data points
  - Focus states

### Platform Color Usage
- Use platform-specific colors for:
  - Platform badges and indicators
  - Chart segments and visualizations
  - Platform-specific analytics

### Status Color Usage
- Use **Green** for excellent/published/viral status
- Use **Blue** for good/trending status
- Use **Orange** for needs improvement/warning status
- Use **Gray** for pending/inactive status

### Background Color Usage
- Use **Very Dark Gray** (`#0A0A0A`) for main background
- Use **Dark Gray** (`#141414`) for cards and containers
- Use **Medium Dark Gray** (`#1F1F1F`) for secondary elements
